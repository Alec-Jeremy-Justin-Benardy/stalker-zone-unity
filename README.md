# stalker-zone-unity
A S.T.A.L.K.E.R. Anomaly inspired Co-op multiplayer game built in Unity 6. Recreating A-Life simulation, Zone atmosphere and authentic STALKER feel — from the ground up in C#. Solo dev project by AJ Dragon Works.
